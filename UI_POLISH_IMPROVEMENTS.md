# UI/UX Polish - Enterprise SaaS Improvements

## Design System Standards

### Spacing Scale
- **xs**: 0.25rem (4px)
- **sm**: 0.5rem (8px)
- **md**: 1rem (16px)
- **lg**: 1.5rem (24px)
- **xl**: 2rem (32px)
- **2xl**: 3rem (48px)

### Typography
- **Headings**: Inter, 700 weight, -0.02em letter-spacing
- **Body**: Inter, 400-500 weight, 1.65 line-height
- **Labels**: Inter, 600 weight
- **Small text**: 0.875rem (14px)

### Colors
- **Primary**: #2563eb (blue-600)
- **Primary Dark**: #1d4ed8 (blue-700)
- **Surface**: #ffffff
- **Surface Muted**: #f8fafc
- **Border**: #e2e8f0
- **Text Primary**: #0f172a
- **Text Secondary**: #64748b
- **Success**: #10b981
- **Warning**: #f59e0b
- **Error**: #ef4444

### Shadows
- **Subtle**: 0 1px 2px 0 rgba(0, 0, 0, 0.05)
- **Card**: 0 1px 3px 0 rgba(0, 0, 0, 0.1), 0 1px 2px -1px rgba(0, 0, 0, 0.1)
- **Elevated**: 0 4px 6px -1px rgba(0, 0, 0, 0.1), 0 2px 4px -2px rgba(0, 0, 0, 0.1)
- **Floating**: 0 10px 15px -3px rgba(0, 0, 0, 0.1), 0 4px 6px -4px rgba(0, 0, 0, 0.1)

### Border Radius
- **Small**: 0.5rem (8px)
- **Medium**: 0.75rem (12px)
- **Large**: 1rem (16px)
- **Full**: 9999px
